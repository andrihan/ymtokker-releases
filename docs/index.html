YMTokker Downloader

Téléchargeur vidéo de bureau — YouTube, playlists, ~1000 sites, torrents. Gratuit, pour Linux et Windows.

⬇ Linux (.deb)Debian / Ubuntu — 64 bits ⬇ Windows (.zip)64 bits — décompresser & lancer
Dernière version sur la page des releases

Sommes de contrôle SHA-256 fournies avec chaque fichier.

Fonctionnalités

Toutes les sources

YouTube, Facebook, Instagram, TikTok, Vimeo… et torrents / liens magnet.

Qualité au choix

De 360p à la meilleure disponible, ou extraction audio MP3 avec tags et pochette.

Playlists & recherche

Sélection des vidéos d'une playlist, recherche YouTube intégrée.

Robuste

Téléchargements en parallèle, pause / reprise, reprise après coupure.

Sous-titres

Export .srt dans la langue de votre choix.

Multilingue

Interface en français, anglais et malgache. Thème clair / sombre.

Installation

Linux — aucune dépendance externe à installer :

sudo apt install ./ymtokker-downloader_<version>_amd64.deb

Windows — décompressez l'archive .zip et lancez ymtokker-downloader.exe.
